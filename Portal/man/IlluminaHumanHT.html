IlluminaHumanHTR Documentation

Illumina HumanHT-12 v4 annotation data

Description

This dataset contains annotation data for the Illumina HumanHT-12 v4 BeadChip gene expression array platform.

Usage

IlluminaHumanHT <- xRDataLoader(RData='IlluminaHumanHT')

Value

a list with 2 components:

  • Info: a data frame. Notably, the column 'Probematch' of 1 indicates a probe sequence is uniquely mapped to genomic location

  • GR: an object of class "GRanges", storing only those probes uniquely mapped to autosomal genomic locations

Examples

IlluminaHumanHT <- xRDataLoader(RData='IlluminaHumanHT')
str(IlluminaHumanHT)
IlluminaHumanHT$Info[1:10,]
IlluminaHumanHT$GR
