JKnc_neutroR Documentation

Table S1 and S3 for cis-eQTL and trans-eQTL in neutrophils from Naranbhai et al. (2015)

Description

This dataset includes cell typeâ€“specific cis and trans expression quantitative trait loci (eQTLs) identified in neutrophils.

Usage

JKnc_neutro <- xRDataLoader(RData.customised='JKnc_neutro')

Value

a list with two entries, 'cis' for cis-eQTLs, 'trans' for trans-eQTLs. Each entry is a data frame with the following columns: "SNP" (dbSNP), "Gene" (gene symbol), "BETA", "T" (test statistics), "P" (p-values), "FDR" (FDR).

References

Naranbhai et al. (2015). Genomic modulators of gene expression in human neutrophils. Nature Communications, 7(6):7545.
