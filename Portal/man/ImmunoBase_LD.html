ImmunoBase_LDR Documentation

Linkage Disequilibrium (LD) of ImmunoBase lead SNPs according to the 1000 Genomics Project data (phase 3)

Description

This dataset involves LD of ImmunoBase SNPs calculated by PLINK based on the 1000 Genomics Project data (phase 3).

Usage

ImmunoBase_LD <- xRDataLoader(RData.customised='ImmunoBase_LD')

Value

a list of sparse Matrix of class "dgCMatrix". Each list represents one of 5 super populations, containing sparse matrix of ImmunoBase lead SNPs (rows) and LD SNPs (columns). Non-zero values indicate r2 (only those no less than 0.8). The population include 5 super populations ("AFR", "AMR", "EAS", "EUR", "SAS"). Explanations for these super-population code can be found at http://www.1000genomes.org/faq/which-populations-are-part-your-study.

References

Malone et al. (2010) Modeling sample variables with an Experimental Factor Ontology. Bioinformatics, 26(8):1112-8.

Examples

ImmunoBase_LD <- xRDataLoader(RData.customised='ImmunoBase_LD')
names(ImmunoBase_LD)
