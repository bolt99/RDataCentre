JKng_monoR Documentation

Table S1 and S2 for cis-eQTL and trans-eQTL in monocytes from Benjamin et al. (2012)

Description

This dataset includes cell type–specific cis and trans expression quantitative trait loci (eQTLs) identified in monocytes.

Usage

JKng_mono <- xRDataLoader(RData.customised='JKng_mono')

Value

a list with two entries, 'cis' for cis-eQTLs, 'trans' for trans-eQTLs. Each entry is a data frame with the following columns: "SNP" (dbSNP), "Gene" (gene symbol), "BETA", "T" (test statistics), "P" (p-values).

References

Fairfax et al. (2012). Genetics of gene expression in primary immune cells identifies cell type–specific master regulators and roles of HLA alleles. Nature Genetics, 44(5):502–510.

Examples

JKng_mono <- xRDataLoader(RData.customised='JKng_mono')
str(JKng_mono)
names(JKng_mono)
JKng_mono$cis[1:5,]
JKng_mono$trans[1:5,]
