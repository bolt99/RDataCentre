GWAS2EFR Documentation

Annotations of GWAS Catalog SNPs by Experimental Factor Ontology (EF)

Description

This dataset involves GWAS Catalog SNPs and their traits mapped to Experimental Factor Ontology. This data is prepared based on https://www.ebi.ac.uk/gwas/api/search/downloads/alternative.

Usage

GWAS2EF <- xRDataLoader(RData='GWAS2EF')

Value

a sparse Matrix of class "dgCMatrix", with the columns for Experimental Factor Ontology terms and rows for GWAS SNPs.

References

Malone et al. (2010) Modeling sample variables with an Experimental Factor Ontology. Bioinformatics, 26(8):1112-8.
Welter et al. (2014) The NHGRI GWAS Catalog, a curated resource of SNP-trait associations. Nucleic Acids Res, 42(Database issue):D1001-6.

Examples

GWAS2EF <- xRDataLoader(RData='GWAS2EF')
GWAS2EF[1:10, 1:33]
